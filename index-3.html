GETTING STARTED

AYYO A.I. Bots act as virtual players in the AYYO game. Bots subscribe and process game state in real-time and respond by sending controller input sequences to the AYYO server.

To create an AYYO AI bot, you'll need to understand AYYO game mechanics and be familiar with any modern programming language.

INSTALL DEVELOPER SDK

AYYO Bot Developer Kits are currently in early access testing.

To install the AYYO SDK for JavaScript run:

npm install @ayyo.ai/sdk
